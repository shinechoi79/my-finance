☁️ Google Sheets 연동

아래에 Google Apps Script 웹앱 URL을 입력하면 PC·모바일 어디서든 데이터가 자동으로 동기화됩니다.

📋 설정 방법
  1. Google Sheets에서 새 스프레드시트 생성
  2. 상단 메뉴 → 확장 프로그램 → Apps Script
  3. 편집기에 google-apps-script.js 내용 붙여넣기
  4. 상단 배포 → 새 배포 → 유형: 웹 앱
  5. 실행 계정: 나, 액세스 권한: 모든 사용자 → 배포
  6. 생성된 URL을 위 입력창에 붙여넣고 저장
